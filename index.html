RallyPoint

table tennis matchmaking and forum

Once users login, they can add a new match, join an open match, view the scoreboard, and discuss in the forum. When adding a new match, the creator sets the time and location of the game. The creator can choose another registered user as an opponent, or select -Open- if the creator would like to leave the match open for other users to join. Users can then view match details such as the location on a map generated by Google Maps API. Once the match has been played, the players enter the scores for the games. The players’ stats are updated on the global scoreboard.

Features: Login and registration with error messages, forum with two separate discussion channels, Google Maps API for match location and directions, Global Scoreboard to track players stats

  • Date: May 2018
  • Technologies: HTML, CSS, C#, ASP.NET, Entity Framework, SQL

MultiGame

real-time tic-tac-toe game and chat

Once users login, they are able to view a list of available games, join or create a game, and play with other users. User scores are logged after each section, and a scoreboard keeps track of player rank. Scoreboard can be sorted either by percentage or number of wins. I was especially involved in login and registration, game logic, and AWS deployment.

Features: Angular framework and socket technology creates a seamless and interactive experience, login and registration with dynamically updating error messages, Users can create games or view and join open games, scoreboard displays player ranking across multiple dimensions, chat feature during play

  • Date: April 2018
  • Technologies: HTML, CSS, JavaScript, Express, Angular, MongoDB, Bootstrap

Spotify Clone

music player using Napster API

Spotify imitation site with functional registration / login, user playlists, and music search. The Napster API is used to pull music information and track samples.

Features: Login and registration with error messages, user information and liked songs stored in database, music information and sample playback from Napster API, track page displaying song information and similar artists, user page displaying all liked songs, artist page displaying popular tracks, dynamic search bar groups results by artist as user types, Semantic UI for styling

  • March 2018
  • Technologies:HTML, CSS, Python, JavaScript, SQL
